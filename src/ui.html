Select a Frame in Figma and click "Export" to convert it into the AI-to-UI JSON format. You can then paste this JSON into an LLM to understand, modify, or extend your design.

💡 Tips for LLM Usage

• Ask for modifications: "Change the primary color to blue and increase padding"

• Request additions: "Add a footer section with social media links"

• Get explanations: "Explain the layout structure of this design"

• Create variations: "Generate a dark mode version of this component"

📐 Node Types

Type Description
FRAME Container with Auto Layout support
TEXT Text layer with font properties
IMAGE Image fill (supports keywords or URLs)
RECTANGLE Shape with fills, strokes, effects
ICON Vector icon from Phosphor library
CHART Data visualization (pie, donut, bar, line)

⚙️ Layout Properties

Property Values
layoutMode "HORIZONTAL" | "VERTICAL"
primaryAxisAlignItems "MIN" | "MAX" | "CENTER" | "SPACE_BETWEEN"
counterAxisAlignItems "MIN" | "MAX" | "CENTER"
itemSpacing number (px)
padding { top, right, bottom, left }
width "fill" | "hug" | number
height "fill" | "hug" | number
layoutPositioning "AUTO" | "ABSOLUTE" (for overlays)
constraints { horizontal, vertical }: "MIN" | "MAX" | "CENTER"

🎨 Styling Properties

Property Values

🎯 Icon Properties

Property Values
icon Phosphor icon name OR raw SVG path string
size number (px, default 24)
color "#HEX"

📊 Chart Properties

Property Values
chartType "PIE" | "DONUT" | "BAR" | "LINE"
chartData [{ value: number, label?: string, color?: "#HEX" }]
donutHoleSize 0-1 (for DONUT charts, default 0.5)

Example: { "type": "CHART", "chartType": "PIE", "chartData": [{ "value": 30, "label": "Sales" }, { "value": 70, "label": "Costs" }], "width": 200, "height": 200 }

🗺️ Sitemap Properties

Property Values
sitemapData Root node object (see example)
nodeWidth number (width of cards, default 180)
levelSpacing number (vertical gap, default 60)
siblingSpacing number (horizontal gap, default 20)

Example: { "type": "SITEMAP", "sitemapData": { "label": "Home", "children": [...] } }

🎲 Fake Data Placeholders

Category Placeholders

Address {{address}}, {{city}}, {{country}}, {{zip}}

Example: "Hello, {{name}}!" → "Hello, Sarah Johnson!"
